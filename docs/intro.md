# Welcome to Cazpian

Cazpian is an AWS-first Lakehouse platform designed for Apache Iceberg and Spark. It provides a powerful environment for data engineering, AI agents, and scalable data management.

## Key Goals
- Simplify data lake management.
- Provide high-performance compute with Apache Spark.
- Enable AI-driven data automation.
