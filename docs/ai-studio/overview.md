# AI Studio Overview

Cazpian AI Studio is the command center for building intelligent data applications.

## What is AI Studio?
It is a low-code/no-code environment where you can connect your Lakehouse data to Large Language Models (LLMs) to create autonomous agents.

### Benefits
- **Direct Data Access**: Agents can query Iceberg tables directly.
- **Scale**: Run agents on top of Spark compute.
- **Automation**: Automate data cleanup, summarization, and reporting.
