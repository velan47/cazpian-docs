# What is Cazpian?

Cazpian is an integrated data platform built for the modern cloud. It leverages open-source standards like Apache Iceberg and Apache Spark to offer a unified Lakehouse experience on AWS.

## Why Cazpian?
Traditional data silos make it hard to innovate. Cazpian breaks down these silos by providing a single point of control for your data assets, compute, and AI workflows.
