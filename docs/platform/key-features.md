# Key Features

Cazpian provides a suite of tools for data-driven organizations:

- **Lakehouse Catalog**: Managed Apache Iceberg catalog for ACID transactions on S3.
- **Serverless Compute**: Integrated Apache Spark engine for high-speed ETL and analytics.
- **AI Studio**: Build and deploy AI agents that interact with your data.
- **Unified Billing**: Simple, transparent pricing based on usage.
- **Data Governance**: Fine-grained access control and auditing.
